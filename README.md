# TMatrix
Matrix representation in linux term just as in "The Matrix" film. At this moment available only for Linux.

# How to use?

`make`

`./tmatrix`

# Install:
`sudo make install`

# Run TMatrix locally
`make`

`./tmatrix`

# Help menu

`./tm --help`

# Debug TMatrix

`make debug`

`gdb dtm`

# Clean-up
`make clean`
